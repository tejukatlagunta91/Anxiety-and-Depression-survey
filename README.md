# Anxiety and Depression Survey Data Analysis

This project uses Machine Learning techniques to analyze and predict anxiety and depression indicators using public health survey data. The project is implemented using Python and Jupyter Notebook.

## Project Overview
The main objective of this project is to identify mental health risk patterns based on demographic and survey-related features. Different classification algorithms were trained and compared to determine the most effective prediction model.

## Dataset
- Dataset: Indicators of Anxiety or Depression
- Source: Kaggle / U.S. Census Bureau Household Pulse Survey
- Records: 16,000+ entries
- Features: Demographic and mental health indicators

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithms
The following models were implemented and compared:
- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Training
7. Model Evaluation
8. Prediction and Visualization

## Model Performance
| Model | Accuracy |
|-------|-----------|
| Logistic Regression | 89.62% |
| SVM | 86.70% |
| Random Forest | 78.63% |

## Features
- Data preprocessing pipeline
- One-Hot Encoding
- Model comparison
- Confusion Matrix visualization
- Accuracy analysis
- Prediction system

## Results
The Logistic Regression model achieved the best performance with an accuracy of 89.62%, making it the final selected model for prediction.
